# React e Express

## Rodar rapidão

``` bash
# Instalar as dependências do server
npm install

# Instalar as dependências do client
npm run client-install

# Rodar o servidor e o cliente ao mesmo tempo
npm run dev

# Roda so o server
npm run server

# Roda so o client
npm run client

# Servidor roda no localhost:5000 e o client no localhost:3000
```
